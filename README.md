# XCPlugin Migration

Make your old Xcode plugins be available on the latest Xcode.

## How to install

```shell
$ git clone https://github.com/sakiwei/xcplugin-migration.git xcplugin-migration
$ cd xcplugin-migration
$ rake build
$ rake install
```

## Usage

```shell
$ xcplugin-migration
```
